# Swappit Consulting LTD — Portfolio Website

A clean, dark-mode portfolio site for **Swappit Consulting LTD**, a technology consulting and product engineering firm based in Lagos, Nigeria.

## Stack

- Pure HTML5, CSS3, Vanilla JS (no frameworks, no dependencies)
- Google Fonts: Syne (display) + Space Grotesk (body)
- Deployed via GitHub Pages / Vercel / Netlify

## Structure

```
swappit-consulting/
├── index.html
├── css/
│   ├── style.css
│   └── animations.css
├── js/
│   └── main.js
├── assets/         ← place logo/favicon here
└── README.md
```

## Features

- Fixed glassmorphic navbar with scroll effect
- Animated hero with live ticker
- Stats band
- Services grid (6 services)
- Portfolio section with browser mockups
- 4-step process timeline
- Contact form
- Fully responsive (mobile-first)
- Scroll-reveal animations
- Reduced-motion support

## Deploy to Vercel

```bash
npm i -g vercel
vercel --prod
```

## Deploy to GitHub Pages

1. Push to `main` branch
2. Go to Settings → Pages → Source: `main` / `root`
3. Site live at `https://<username>.github.io/swappit-consulting`
